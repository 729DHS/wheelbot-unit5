这份文档是 **《RoboMaster 开发板 C 型用户手册 v1.0》**，由大疆创新（DJI）于2020年发布。

这份手册详细介绍了 RoboMaster 开发板 C 型的硬件结构、接口定义、电气特性和使用方法。为了帮助你快速掌握核心信息，我将文档内容梳理为以下几个关键模块：

### 1. 核心概述与清单
*   **简介**：该开发板基于 **STM32F407** 主控芯片，支持 8V-28V 宽电压输入。它集成了扩展接口、通信接口以及高精度 IMU 传感器，主要用于机器人比赛、科研教育等领域。
*   **物品清单**：
    *   开发板 C 型 ×1
    *   SWD 下载线 ×1
    *   电源线 ×1
    *   2-Pin 线 ×1
    *   4-Pin 线 ×1

### 2. 接口与线序（关键硬件信息）
开发板集成了丰富的接口，以下是主要接口的规格：

| 接口类型 | 数量/规格 | 说明 |
| :--- | :--- | :--- |
| **电源接口** | XT30 | 红色为正(A)，黑色为负(B)；具备防反接和过压保护(>28V关断) |
| **CAN 总线** | 2路 | CAN1为2-pin (L/H)；CAN2为4-pin (5V/GND/H/L)，支持1M速率 |
| **UART** | 2路 | UART1为4-pin；UART6为3-pin；仅支持3.3V/5V电平 |
| **PWM** | 7路 | 最大总电流 5A，用于连接舵机或驱动模块 |
| **SWD** | 1路 | 用于程序下载和调试 (SWDIO/SWCLK/GND/3.3V) |
| **其他接口** | 多种 | 包括 I2C/SPI 扩展口、DBUS 遥控接口、数字摄像头 FPC 接口等 |

### 3. 核心功能模块
*   **电源系统**：
    *   采用 TPS54540 等芯片进行电压转换。
    *   提供 24V 转 5V（两路，分别用于外设和板载）以及 5V 转 3.3V 电路。
*   **传感器**：
    *   **六轴 IMU**：采用 **BMI088** 芯片，具备特殊减震结构和加热电路（通过 TIM10_CH1 控制），以减少温漂。
    *   **磁力计**：采用 **IST8310** 三轴磁力计，I2C 地址默认 0x0E。
*   **人机交互**：
    *   **LED**：1 颗共阳极 RGB 三色灯（PH10/11/12 控制）。
    *   **按键**：1 个复位键 (RST) 和 1 个用户自定义键 (PA0)。
    *   **蜂鸣器**：贴片无源蜂鸣器，需 4kHz PWM 驱动。

### 4. 固件下载与启动模式
开发板支持两种主要的程序下载方式：
1.  **SWD 模式**：使用 J-Link 或 ST-Link 仿真器通过 SWD 接口下载，适合日常开发调试。
2.  **DFU 模式 (USB下载)**：
    *   需配置 **BOOT0=1, BOOT1=0**。
    *   通过 Micro-USB 接口连接电脑。
    *   使用 Dfu File Manager 和 DfuSe Demo 软件进行固件更新。

### 5. 物理规格与环境
*   **尺寸**：60.00 × 41.00 × 16.3 mm
*   **重量**：38g
*   **安装孔**：4个，内径 2.5mm，外径 5.0mm
*   **工作环境**：温度 0-55℃，待机电流 0.01A@24V

如果你需要进行具体的开发（如配置某个引脚或连接特定外设），建议查阅手册中的“附表：单板网络名与IO对照表”以获取精确的 GPIO 映射信息。

### 📊 对应关系对照表
为了方便你接线和编程，我将文档中的对应关系整理如下：

| 外壳丝印 (板子上标的) | STM32 内部实际接口 | 备注 |
| :--- | :--- | :--- |
| **UART1** | **UART6** | 物理接口为 3-pin |
| **UART2** | **UART1** | 物理接口为 4-pin |

### 💡 总结
这意味着，当你在代码中配置串口时：
*   如果你要使用板子上标着 **UART1** 的那个 3-pin 接口，你应该在程序中初始化 **USART6**。
*   如果你要使用板子上标着 **UART2** 的那个 4-pin 接口，你应该在程序中初始化 **USART1**。

建议在开发时将上述对照表作为参考，以免接错外设或配置错中断服务函数。
